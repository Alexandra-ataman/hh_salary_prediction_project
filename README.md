# Проект: Анализ резюме из HeadHunter

Проект по анализу данных резюме соискателей с HeadHunter для прогнозирования заработной платы.

## Структура проекта

- `Python_для_анализа_данных__Итоговый_проект_Ноутбук-шаблон.ipynb` - основной ноутбук с анализом
- `graphs/` - папка с интерактивными графиками в формате HTML
